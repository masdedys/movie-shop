DateTime example Date and time

This example shows the date / time input used for both date and time inputs.

The JavaScript shown below is used to initialise the table shown in this example:

$('#test').dtDateTime({ format: 'D MMM YYYY HH:mm' });
new DateTime(document.getElementById('test'), { format: 'D MMM YYYY HH:mm' });
