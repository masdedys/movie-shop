SearchBuilder example Using Orthogonal Data

This example shows how SearchBuilder can make use of orthogonal data.

The "Rating" column renders a progress bar depending on the value that it holds. SearchBuilder is unable to display progress bars in it's select elements. By setting columns.searchBuilder.orthogonal to have an orthogonal.display value of 'filter', the raw numerical data is displayed instead.
